GetQuote

Up 
get /v1.0/{chainId}/getQuote
Fetches the best aggregated quotes for given trading pair

Path parameters

chainId (required)
Path Parameter —

Query parameters

buyTokenAddress (required)
Query Parameter —
skipValidation (required)
Query Parameter —
sellTokenAmount (optional)
Query Parameter — Add one of (sellTokenAmount, buyTokenAmount)
buyTokenAmount (optional)
Query Parameter — Add one of (sellTokenAmount, buyTokenAmount)
fromAddress (required)
Query Parameter —
sellTokenAddress (required)
Query Parameter —
slippage (required)
Query Parameter — Price slippage you are willing to accept, may be set with decimals.
(e.g. slippage=0.005 means 0.5% slippage is acceptable; slippage=0.01 means 1% slippage is acceptable, etc.)
recipient (optional)
Query Parameter — ETH address
affiliate (optional)
Query Parameter — The ETH address that should receive affiliate fees (if specified)
affiliateFee (optional)
Query Parameter — This percentage of (sellTokenAddress(1inch) / buyTokenAddress (0x)) token amount will be sent to affiliate, the rest will be used as for the swap. (e.g. affiliateFee=0.5 equals 0.5%)
signaturePermitData (optional)
Query Parameter —

Return type

Object
